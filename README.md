Reflexión:
Aprender a utilizar la terminal resulta importante debido a que permite trabajar de manera más directa y rápida a través de comandos.
En cambio, mediante la interfaz gráfica, las acciones requieren navegar por menús y ventanas, lo que supone más pasos.
Por otro lado, el uso de la terminal permite clarificar el funcionamiento del sistema.
Incluso en algunos sistemas no se cuenta con interfaz gráfica, por lo que no queda otra opción que el manejo desde la terminal.

10 comandos de terminal utilizados

1.
Comando: git --version
Función: Muestra la versión de Git instalada.
Uso: Para verificar la correcta instalación de Git.

2.
Comando: code --version
Función: Muestra la versión de Visual Studio Code instalada.
Uso: Para verificar la correcta instalación de VSCode.

3.
Comando: pwd
Función: Muestra la ruta de la carpeta actual en la terminal.
Uso: Para saber en qué directorio estaba posicionada.

4.
Comando: mkdir tpl-mi-sitio
Función: Crea una nueva carpeta.
Uso: Para crear la carpeta del sitio web.

5.
Comando: cd tpl-mi-sitio
Función: Permite ingresar a un directorio específico.
Uso: Para ingresar a la carpeta del proyecto.

6.
Comando: mkdir images
Función: Crea una nueva carpeta.
Uso: Para guardar las imágenes del sitio web.

7.
Comando: touch index.html
Función: Crea un nuevo archivo.
Uso: Para crear el archivo html de la página web.

8.
Comando: touch style.css
Función: Crea un nuevo archivo.
Uso: Para crear el archivo de estilos CSS.

9.
Comando: ls
Función: Lista los archivos y carpetas del directorio actual.
Uso: Para corroborar la correcta creación de las carpetas y archivos.

10.
Comando: code .
Función: Abre la carpeta actual en Visual Studio Code.
Uso: Para comenzar a editar el proyecto.

11.
Comando: clear
Función: Limpia la pantalla de la terminal.
Uso: Para dejar vacía la pantalla de la terminal con el fin de trabajar de manera más ordenada.

12.
Comando: cat index.html
Función: Muestra el contenido de un archivo directamente en la terminal.
Uso: Para ver el contenido del archivo index.html sin abrir el editor.
